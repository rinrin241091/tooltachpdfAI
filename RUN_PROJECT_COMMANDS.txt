TOOLTACHPDFAI - HUONG DAN CAI DAT VA CHAY DU AN
==============================================

MUC TIEU
--------
Tai lieu nay gom huong dan:
1) Cai thu vien cho Backend va Frontend
2) Chay local (khong Docker)
3) Chay bang Docker
4) Kiem tra nhanh va fix loi thuong gap tren Windows


I) YEU CAU MOI TRUONG
---------------------
1) Python 3.10+ (khuyen nghi 3.11)
2) Node.js 18+ (khuyen nghi 20)
3) pip moi nhat
4) Docker Desktop (neu muon chay bang Docker)

Cap nhat pip:
python -m pip install --upgrade pip


1) CAI THU VIEN BACKEND
-----------------------
Mo terminal tai thu muc goc du an:

cd backend
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt

Neu can cai thu cong theo nhom:

pip install fastapi==0.111.0
pip install "uvicorn[standard]==0.29.0"
pip install PyMuPDF==1.24.3
pip install paddleocr==2.7.3
pip install paddlepaddle==2.6.1
pip install opencv-python-headless==4.9.0.80
pip install Pillow==10.3.0
pip install numpy==1.26.4
pip install python-multipart==0.0.9
pip install httpx==0.27.0

Ghi chú:
- paddlepaddle la bản CPU. Nếu dùng GPU, đổi sang paddlepaddle-gpu theo CUDA phu hop.


2) CAI THU VIEN FRONTEND
------------------------
-Mở terminal mới:

cd frontend
npm i

Cài them thư viện cho AG Grid + zip + icon:

npm install ag-grid-react ag-grid-community jszip file-saver lucide-react axios

Ghi chu:
- Tailwind v3, postcss, autoprefixer da co trong devDependencies.
- File tailwind.config.js va postcss.config.js da duoc tao san.
- Khong can chay npx tailwindcss init.
- Chi can chay npm.cmd install la du.


3) CAU HINH TAILWIND (DA THIET LAP SAN - KHONG CAN THAO TAC)
-------------------------------------------------------------
Cac file sau da co san trong du an, khong can tao them:
- frontend/tailwind.config.js
- frontend/postcss.config.js
- frontend/src/index.css (da co @tailwind directives)


4) CHAY LOCAL (KHONG DOCKER)
----------------------------
1) Chay Backend FastAPI:

cd backend
python -m venv .venv
.\.venv\Scripts\activate

** BUOC 1 - Nang cap pip:
python -m pip install --upgrade pip

** BUOC 2 - Cai tat ca bang binary wheel (tranh loi build source tren Python 3.13/3.14):
pip install --only-binary=:all: -r requirements.txt

   Neu buoc tren bao loi "no matching distribution" cho bat ky goi nao,
   cai tung goi rieng nhu sau:
   pip install --only-binary=:all: fastapi "uvicorn[standard]" PyMuPDF opencv-python-headless Pillow numpy python-multipart httpx

** BUOC 3 - Chay server:
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload

** LUU Y CHUNG:
- Luon dung python -m uvicorn thay vi goi thang uvicorn (tranh loi "not recognized")
- Luon them --only-binary=:all: khi cai tren Python 3.13+ de tranh pip co build source
- paddlepaddle (OCR) chi ho tro Python <= 3.12, cai rieng bang requirements-ocr.txt
  neu can dung tinh nang OCR (xem muc 1b ben duoi)

** BUOC 1b (TUY CHON) - Cai PaddleOCR neu can chay tinh nang OCR:
   - Yeu cau Python <= 3.12. Neu dang dung Python 3.13/3.14, tao venv rieng:
     py -3.12 -m venv .venv-ocr
     .\.venv-ocr\Scripts\activate
     pip install -r requirements-ocr.txt

2) Mo terminal moi de chay Frontend React:

cd frontend
npm start

3) Truy cap:
- Frontend: http://localhost:3000
- Backend API docs: http://localhost:8000/docs

5) CHAY 1 LENH O WINDOWS
------------------------
- Mo PowerShell tai thu muc goc du an va chay:
  .\run-dev.bat
  hoac
  .\run-dev.ps1
- Script se tu:
  - tao backend\\.venv neu chua co
  - cai backend dependencies neu can
  - chay backend FastAPI port 8000
  - chay frontend React port 3000


5) CHAY BANG DOCKER
-------------------
1) Mo terminal tai thu muc:

cd shared

2) Build + run:

docker compose up --build

3) Truy cap:
- Frontend: http://localhost:3000
- Backend: http://localhost:8000

4) Dung container:

docker compose down


6) LENH KIEM TRA NHANH
----------------------
1) Kiem tra backend endpoint upload/analyze/split:
- Vao http://localhost:8000/docs
- Thu endpoint /upload, /analyze/{file_id}, /split

2) Neu front khong goi duoc API:
- Kiem tra backend dang chay cong 8000
- Kiem tra CORS trong backend/main.py


7) GOI Y FIX LOI THUONG GAP (WINDOWS)
-------------------------------------
1) npm bi chan boi ExecutionPolicy:
- Dung npm.cmd thay cho npm

2) Loi import paddleocr/paddlepaddle:
- Kiem tra dung Python version
- Cai lai requirements:
  pip install -r requirements.txt

3) Loi OpenCV/PaddleOCR:
- Dam bao da cai opencv-python-headless
- Khoi dong lai terminal

4) PaddleOCR chay cham:
- Giam so trang test khi analyze
- Dam bao may co du RAM


8) CAU TRUC THU MUC CHINH
-------------------------
- backend/ : FastAPI + OCR + split PDF
- frontend/: React UI + AG Grid
- shared/  : docker-compose va file lien quan
